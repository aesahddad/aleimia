# Aleinia Sidebar Rules

## Position
- السايد بار دائماً على جهة زر البرقر (اليمين في RTL)
- موقعه لا يتغير أبداً

## Visibility
- **ديسكتوب (>= 768px)**: السايد بار ظاهر دائماً، والبرقر يتحكم في إخفائه/إظهاره
- **جوال (< 768px)**: السايد بار يغطي كامل الشاشة (100% عرض وارتفاع) عند فتحه
- في الجوال: إما السايد بار كامل أو الصفحة الرئيسية — لا يظهر شيء من الثانية

## Page Tabs
- كل صفحة لها أزرارها الخاصة في السايد بار
- زر الصفحة الحالية يتم إخفاؤه من السايد بار
- الأولوية للسايد بار عند التنقل بين الصفحات (لأن أزرار الصفحات فيه)

## Video
- السايد بار يظهر مكان للفيديو الدعائي دائماً (حتى لو مافي رابط)
- الفيديو يكون بحجم شاشة الجوال عامودياً (min-height: 80vh)
- صفحة الاشتراكات لها فيديو خاص (promoVideoPlansUrl)
- باقي الصفحات تستخدم الفيديو العام (promoVideoUrl)

## Content Order
1. الشعار + اسم "فهرس العينية" + "نعطي لحلمك بعداً آخر"
2. الفيديو الدعائي (مع placeholder إذا مافي رابط)
3. أزرار خاصة بالصفحة الحالية
4. معلومات خاصة بالصفحة (إن وجدت)
5. أزرار تسجيل الدخول/المستخدم + لوحات التحكم
